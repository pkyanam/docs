# About me

I'm Preetham Kyanam, a computer engineer based in the US. I graduated from Virginia Tech in May 2023 with a B.S. in Computer Engineering.

I spend my time building open-source software that spans everything from embedded systems and hardware interfaces to AI coding tools and web applications. I'm especially drawn to the intersection of AI and developer tooling — building MCP servers, CLI agents, and other tools that make development faster and more expressive.

## What drives me

I believe in learning by building. Whether it's a Rust-based CLI tool, an Arduino library that controls a Roomba, or an MCP server that wakes up a PS5 — every project is a chance to go deeper into a technology and ship something real.

## Technical interests

- **AI-accelerated development** — MCP servers, coding agents, CLI assistants
- **Embedded systems** — Arduino, hardware-software interfaces
- **Systems programming** — Rust, C++
- **Web development** — TypeScript, Next.js, React
- **Native apps** — Swift, macOS menu bar utilities
