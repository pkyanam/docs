# About me

I'm Preetham Kyanam, a computer engineer based in the US. I graduated from Virginia Tech in May 2023 with a B.S. in Computer Engineering.

I spend my time building open-source software and developer tools that span embedded systems, AI agents, voice interfaces, and web applications. I'm especially drawn to the intersection of AI and developer tooling — building MCP servers, CLI agents, and cloud infrastructure that makes development faster and more expressive.

## What I'm building

**[AgentMeld](https://agentmeld.com)** is a service I run that bridges asynchronous cloud agents with local coding agents, giving developers a way to offload long-running agentic tasks to the cloud while keeping local context. It's early-stage and actively in development.

I also develop **abilities for [OpenHome](https://openhome.ai)**, a voice AI and smart speaker platform. The OpenHome team provided me with a free dev kit to build on, which has been a great way to explore voice-first interaction design.

## What drives me

I believe in learning by building. Whether it's a Rust-based CLI tool, an Arduino library that controls a Roomba, or an MCP server that wakes up a PS5 — every project is a chance to go deeper into a technology and ship something real.

## Technical interests

- **Voice AI** — OpenHome abilities, smart speaker development, voice-first UX
- **Embedded systems** — Arduino, hardware-software interfaces
- **Systems programming** — Rust, C++
- **Web development** — TypeScript, Next.js, React
- **Native apps** — Swift, macOS menu bar utilities
